
# Laravel HLI

Learning yang dibuat menggunakan framework Laravel


## Run Locally

Clone the project

```bash
  git clone https://github.com/
```

Go to the project directory

```bash
  cd laravel_
```

Install dependencies

```bash
  composer install
```

Copy env.example

```bash
  cp .env.example .env
```

Generate key

```bash
  php artisan key:generate
```

Migrate into database

```bash
  php artisan migrate --seed
```

Start the server

```bash
  php artisan serve
```
